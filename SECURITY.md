# Security Policy

## Supported versions

BlazorThree is currently in preview. Security fixes are applied to the latest preview version published on NuGet.

## Reporting a vulnerability

Please do not open public GitHub issues for suspected vulnerabilities.

Use one of these channels instead:

- Open a private GitHub security advisory for this repository
- Contact the maintainer through GitHub and include reproduction details

Please include:

- A description of the issue
- Affected package version
- Steps to reproduce
- Any proof-of-concept or relevant logs

You can expect an acknowledgment after the report is reviewed. Public disclosure should wait until a fix or mitigation is available.
